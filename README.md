# Парсер прайса клиники
Привет! Это мой проект с хакатона.
Я написал код который сам читает PDF файл с ценами на медицинские услуги
и переносит всё это в Excel таблицу. Руками это делать очень долго,
а так всё происходит за несколько секунд.
## Зачем это нужно
Обычно прайс-листы клиник это большие PDF файлы на много страниц.
Найти нужную услугу или сравнить цены неудобно.
Мой код переносит всё в таблицу где можно легко искать и фильтровать.
## Как запустить
1. Зайди на сайт colab.research.google.com
2. Загрузи туда файл с кодом
3. Загрузи свой PDF прайс
4. Нажми запустить и жди
5. Скачай готовый Excel файл
## Что использовал
- Python
- библиотека pdfplumber (читает PDF)
- библиотека pandas (делает таблицы)
